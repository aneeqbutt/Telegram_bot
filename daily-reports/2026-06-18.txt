CHALLENGES RESOLVED

OpenRouter model returning 504 gateway timeout

* Root cause:
  nvidia/nemotron-3-ultra-550b-a55b:free model was cold and
  unresponsive, timing out before returning a response

* Fix:
  Switched AI_MODEL to meta-llama/llama-3.3-70b-instruct:free
  then to current working model -- OpenRouter auto-routed to
  nex-agi/nex-n2-pro:free which responded successfully

pytest-asyncio not recognized on Python 3.12

* Root cause:
  pip install ran against Python 3.11 but pytest was running
  under Python 3.12 -- asyncio_mode = auto had no effect

* Fix:
  Ran python -m pip install pytest-asyncio inside Python 3.12
  environment, all async tests passed immediately

test_classify_title_weighted_higher failing

* Root cause:
  Test content contained the word ethereum which also matched
  the eth keyword via substring, giving Ethereum 2 points vs
  Bitcoin 1 point unexpectedly

* Fix:
  Rewrote test content to avoid any crypto keyword in body,
  confirmed title-only match correctly classified as Bitcoin

MongoDB Atlas SSL handshake failure blocking server startup

* Root cause:
  IP address changed between sessions, current IP not in Atlas
  Network Access allowlist

* Fix:
  Pending -- user needs to add current IP in Atlas Network
  Access before server can start
